This directory holds several Kicad5 pcbs:

- pcbv4.pro         --> main neOSensor board (esp8266 version)
- pcbv4-temp.pro    --> addon board featuring a temperature sensor
- neOSensorV4_panel --> neOSensors panelized ;)

Note: we get inspiration from Arduino MKR1300 schematic for power supply

To launch Kicad: #> kicad xxx.pro
